This project has custom SQL database Source and Sink Kafka Connectors.
If you want to write your own custom connector,Then this project is one example for you
If you want to use custom kafka connector,Please make jar file of this project and place inside libs folder of kafka
After placing as jar Then only you can start this connector
This project is not for standalone run.
